# Splitenry

Split your browser into resizable panels. View multiple websites side-by-side in a single window with draggable dividers.

## Features
- **Single-Tab Tiling:** Keep your workspace clean by loading multiple sites into iframes within one tab.
- **Resizable Panels:** Freely adjust the size of your panels using draggable dividers.
- **Pre-built Layouts:** Choose from 12+ preset layouts (e.g., 2-pane split, 3-pane grid, main + sidebar).
- **Instant Navigation:** Each panel acts as its own browser with a dedicated URL bar.
- **100% Local & Private:** The extension runs entirely on your machine. No user data is collected.

## Installation (Developer Mode)
1. Clone this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** in the top right.
4. Click **Load unpacked** and select the cloned folder.

## Build
Run `make build` to package the extension into a `.zip` file for distribution.
