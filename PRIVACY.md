# Privacy Policy

This Privacy Policy applies to this browser extension.

## 1. Data Collection
This extension **does not** collect, store, or transmit any personal information or user data. 

## 2. Local Processing
All processing (including network modifications for layout tiling or encryption tasks) happens entirely locally on your device within your browser. 

## 3. Storage
If the extension requires saving settings (such as UI layout preferences or offline keys), this data is stored strictly in your browser's local storage (`chrome.storage.local`). It is never synced to external servers or accessible by the developer.

## 4. Third-Party Services
This extension does not use any third-party analytics, tracking, or telemetry services.

## Contact
If you have any questions about this Privacy Policy, please open an issue on this GitHub repository.
